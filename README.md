BoostX — Full Project (Balance + Looksmm Integration)
====================================================

1) Укажи переменные в .env (TOKEN, ADMIN_ID уже есть шаблон).
2) На Render добавь те же env (или загрузку .env).
3) Deploy: Manual Upload или git push.

Команды:
/start — меню категорий
/balance — баланс
/topup <сумма> — создать пополнение (уникальная сумма + код)
/pay — ссылка и инструкция
/paid — отправить заявку на подтверждение оплаты
/reply <user_id> <текст> — ответ пользователю (для ADMIN_ID)
/whoami — показать ID
/promo_* — управление промокодами

Заказ:
Категория → товар → количество (мин. 10) → ссылка → списание с баланса → заказ поставщику (Looksmm).

Конфиг цен/товаров: config/config.json (можно менять без перекомпиляции).
